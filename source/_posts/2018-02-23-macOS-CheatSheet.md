homebrew
=========

```bash
brew doctor
brew update
brew upgrade

brew search wget
brew info wget
brew install wget
brew list

brew cask search iterm2
brew cask info iterm2
brew install iterm2
brew cask list
```

nvm
======

```bash
nvm ls-remote
nvm install v8.9.4
nvm use v8.9.4
nvm current
nvm ls
nvm alias default v8.9.4
```

hexo
======

```bash
hexo new "Post Title"
hexo generate
hexo deploy
hexo clean
hexo server
```
